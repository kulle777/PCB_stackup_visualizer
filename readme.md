# PCB Stackup Visualization App

Created by **Kalle Paasio**

## Purpose
This Streamlit app was created to help in PCB design by visualizing stackups and layer thicknesses in an easy-to-understand format.

Reason for creation is that no other app has this feature.

## Features
- Enter custom layer thickness values.
- Works on any unit
- Download the generated stackup image as a PNG file.
